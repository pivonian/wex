The Wex Framework

What is Wex?

Wex is Doubt's web development framework designed to streamline the process of building and maintaining websites specifically streaming sites (e.g. ViewPoint). It combines modern web technologies with best practices to ensure fast, responsive, and scalable web applications.

How to Use Wex

Wex is currently in alpha and is being actively developed. It will be available soon in 2 versions the first "Wex for Enterprise" being the full version of Wex and "Wex Community" being a lighter version for individual developers and small teams.

Pricing

Wex for Enterprise will be available soon for A$159.99 which permits the usage of Wex for commercial projects with full support and updates.

Wex Community will be available for free for non-commercial projects with community support. Since it will be Open-Source it users will be permitted to fork & modify this version of Wex.
